# Tech Stack: Brincar Educando

## Frontend
- **Framework**: Next.js 16.1.6 (App Router)
- **Linguagem**: TypeScript
- **Estilização**: Tailwind CSS 4, CSS Modules, Framer Motion (animações)
- **Componentes UI**: Radix UI, Lucide React, Embla Carousel, Recharts, Shadcn/UI
- **Gerenciamento de Estado**: Zustand
- **Formulários**: React Hook Form + Zod

## Backend & Data
- **Backend-as-a-Service**: Supabase (Auth, Database, Storage)
- **Client**: @supabase/supabase-js, @supabase/ssr
- **Data Fetching**: TanStack Query (React Query) v5
- **API**: Next.js API Routes (Route Handlers)

## Content
- **Blog**: MDX (next-mdx-remote, gray-matter)
- **CMS Portátil**: Arquivos locais em `content/` e metadados no Supabase.

## DevOps & Tools
- **Deployment**: Vercel (provavelmente)
- **Linting**: ESLint
- **Package Manager**: npm
- **Cerebro**: Conductor CLI / Antigravity Agent
